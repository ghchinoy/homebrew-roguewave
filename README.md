# Rogue Wave Homebrew Tap

This repository is a convenience for installing my cli tools:

* [rwctl](https://github.com/ghchinoy/rwctl), a cli for the Rogue Wave API Platform

## Installation

```
brew update
brew install ghchinoy/akana/atmotool
```
